## ✅ Como usar

1. Instale [BetterDiscord](https://betterdiscord.app/) ou [Vencord](https://vencord.dev/)
2. Baixe o arquivo `.theme.css` em `themes/`
3. Copie para sua pasta de temas
4. Ative nas configurações do app
